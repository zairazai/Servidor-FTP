# Práctica: Servidor FTP

## Descripción
Implementación de un servidor FTP en Linux (vsftpd) y Windows Server (IIS FTP), con acceso anónimo y autenticación por grupos.

## Entorno
- Ubuntu Server
- Windows Server
- Ubuntu Desktop (cliente)

## Objetivo
Automatizar la configuración del servicio FTP incluyendo usuarios, grupos, permisos y acceso controlado.
